Your Stack
⚙️
BPMN Engine Camunda 7
BPMN engine with REST API. Design processes in the built-in modeler and deploy with one click.
📋
Log Viewer Dozzle
Real-time Docker log viewer. Stream and filter logs from all containers in one place.
🐘
Database Browser pgweb
Browse and query your database directly in the browser. Inspect tables, run SQL, export data.
✏️
BPMN Modeler Miragon
Design BPMN processes in the browser and deploy to Camunda with one click. No desktop app needed.
🐳
Container Manager Portainer
Monitor and manage your Docker containers. Restart services, view resource usage, check health.
Did You Know?
💡
Access all your services from one place
Studio runs everything locally. The BPMN Engine at :8080, Log Viewer at :8082, Database Browser at :8081, Container Manager at :9443, and the BPMN Modeler is built right into Studio. All accessible from the sidebar.
